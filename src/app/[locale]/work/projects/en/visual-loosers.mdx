## ¿Que es Visual Loosers?

Es un generador de visuales a partir de código GLSL. Esto quiere decir que cuando entres en la página veras un editor de texto en donde podrás escribir código y este se renderizara cuando toques el botón "Ejecutar código". Este código sera ejecutado en el Canvas que forma el display del resto de la aplicación. ¿Simple no? Puedes empezar a ver resultados con el botón Random, el cual te "genera" códigos aleatorios (ya preestablecidos generados con IA) para que puedas empezar a familiarizarte con la interfaz de la aplicación.
VisualLoosers está escrito en JavaScript e implementa WebGL a traves de p5.js.

## Key Features

- **Time**: Un valor flotante que representa el tiempo en segundos desde que comenzó la ejecución. Útil para animaciones.
- **resolution**: Un vector de dos componentes (vec2) que representa el tamaño del canvas en píxeles. Se usa para normalizar coordenadas y adaptar el visual a la resolución de la pantalla.
- **gl_FragCoord.xy**: Coordenadas del fragmento actual en el espacio de pantalla. Combinadas con resolution, permiten crear efectos basados en la posición.
- **length**: Calcula la distancia desde el origen, útil para efectos radiales.

- **JavaScript**: Para la interactividad con el editor de texto.
- **CSS**: Estilizar el editor de texto y la web.
- **WebGL y p5.js**: Para la modificación de los shaders del canvas con GLSL.
- **GitHub Pages**: Para hostear la página.
